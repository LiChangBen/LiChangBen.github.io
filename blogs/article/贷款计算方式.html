贷款计算方式

LiChangBen 2021-11-07 知识

# 贷款计算方式

# 两者还款的计算方式：

（一）等额本息还款：

每月还本付息=【贷款本金x月利率x（1+月利率）还款月数】/【（1+月利率）还款月数-1】

每月还利息=剩余本金x贷款月利率（第一个月的本金就是贷款额，逐月递减）

每月还款本金=每月还款金额-每月还利息

剩余本金=上月剩余本金-本月偿还本金

还款总利息=贷款额X贷款月数x月利率x（1+月利率）还款月数/【（1+月利率）还款月数-1】-贷款总额

（二）等额本金还款

每月还本付息=（本金/还款月数）+（本金-累计已还本金）x月利率

每月本金=总本金/还款月数

每月利息=（本金-累计已还本金）x月利率

还款总利息=（还款月数+1）x贷款额x月利率/2

提示

贷款的计算方式都是一样的，只是商业贷款的利率比公积金贷款的利率高
组合贷款 每月还款金额 是商业贷款和公积金贷款总和，

计算的时候尽量精确,保留的小数位尽量多点,算出来的结果才会准确

计算方法

   /**
     * 等额本金 计算还款信息
     * @param totalPrice 总金额
     * @param rate 利率
     * @param month 贷款总月份
     * @return
     */
    private List<Map<String,Object>> eqFundCalcLoan(double totalPrice, double rate, int month) {
        //等额本金
        List<Map<String,Object>> eqFund = new ArrayList<>();
        // 累计还款本金
        double  totalFund =0.0;
        for(int i=0; i<month; ){
            Map<String,Object> monthMap = new HashMap<>();
            i++;
//            int curMonth = i; //当前月
            monthMap.put("curMonth",i);
            //每月本金=总本金/还款月数
            double principal = NumberUtils.divide(totalPrice,month,30);
            monthMap.put("principal",new BigDecimal(Double.toString(principal)).setScale(2, BigDecimal.ROUND_HALF_UP).doubleValue());//
            //每月利息=（本金-累计已还本金）x月利率
            double interest = NumberUtils.multiply(NumberUtils.subtract(totalPrice,totalFund),rate);
            monthMap.put("interest",new BigDecimal(Double.toString(interest)).setScale(2, BigDecimal.ROUND_HALF_UP).doubleValue());
            //每月还本付息=（本金/还款月数）+（本金-累计已还本金）x月利率
            //每月还本付息 = 每月本金+每月利息
            double monthTotalPrincipal = NumberUtils.add(principal,interest);
            monthMap.put("monthTotalPrincipal",new BigDecimal(Double.toString(monthTotalPrincipal)).setScale(2, BigDecimal.ROUND_HALF_UP).doubleValue());
            totalFund = NumberUtils.add(totalFund,principal);
            //剩余本金
            monthMap.put("oddRorpus",new BigDecimal(Double.toString(NumberUtils.subtract(totalPrice,totalFund))).setScale(2, BigDecimal.ROUND_HALF_UP).doubleValue());
            eqFund.add(monthMap);
        }
        return eqFund;
    }

    /**
     * 等额本息 计算还款信息
     * @param totalPrice 总金额
     * @param rate 利率
     * @param month 贷款总月份
     * @return
     */
    private List<Map<String,Object>> eqAccrCalcLoan(double totalPrice, double rate, int month) {
        List<Map<String,Object>> eqAccr = new ArrayList<>();//等额本息
        // 剩余本金
        double  awaitPrincipal = totalPrice;
        for(int i=0; i<month; ){
            Map<String,Object> monthMap = new HashMap<>();
            i++;
//            int curMonth = i; //当前月
            monthMap.put("curMonth",i);
            // 每月还利息 = 剩余本金 x 贷款月利率（第一个月的本金就是贷款额，逐月递减）
            double interest = NumberUtils.multiply(awaitPrincipal,rate);
            monthMap.put("interest",new BigDecimal(Double.toString(interest)).setScale(2, BigDecimal.ROUND_HALF_UP).doubleValue());
            // 每月还本付息 =【贷款本金 x 月利率x（1+月利率）^还款月数】/【（1+月利率）^还款月数-1】
            double monthTotalPrincipal = NumberUtils.divide(
                    NumberUtils.multiply(NumberUtils.multiply(totalPrice,rate),this.pow(NumberUtils.add(rate,1),month,20)),
                    NumberUtils.subtract(this.pow(NumberUtils.add(rate,1),month,20),1),20);
            monthMap.put("monthTotalPrincipal",new BigDecimal(Double.toString(monthTotalPrincipal)).setScale(2, BigDecimal.ROUND_HALF_UP).doubleValue());
            // 每月还款本金 = 每月还款金额-每月还利息
            double principal = NumberUtils.subtract(monthTotalPrincipal,interest);
            monthMap.put("principal",new BigDecimal(Double.toString(principal)).setScale(2, BigDecimal.ROUND_HALF_UP).doubleValue());
            // 剩余本金
            monthMap.put("oddRorpus",new BigDecimal(Double.toString(NumberUtils.subtract(awaitPrincipal,principal))).setScale(2, BigDecimal.ROUND_HALF_UP).doubleValue());
            eqAccr.add(monthMap);

            awaitPrincipal = NumberUtils.subtract(awaitPrincipal,principal);
        }
        return eqAccr;
    }

    private double pow(double num1, int num2, int scale) {
        if (scale < 0) {
            throw new IllegalArgumentException(" 截取位数必须是正整数或者零 ");
        }
        BigDecimal big1 = new BigDecimal(Double.toString(num1));
        return big1.pow(num2).setScale(scale, BigDecimal.ROUND_HALF_UP).doubleValue();
    }
